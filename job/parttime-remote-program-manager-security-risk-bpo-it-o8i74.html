Program and CS Tools Support Management (PSM), a key component of the CS Technology team, manages technical issue escalations and champions enhanced reliability for CS tools and technologies. In this role, you will manage security and risk prevention programs for Customer Service in partnership with stakeholders from the broader Netflix Security Organization. Success in this role means you will have built strong collaborative relationships, effectively mitigated risks, ensured compliance with security standards, and implemented robust programs that consistently enhance the security and quality of the Customer Service Organization. Core Responsibilities:  Define and manage the security program within Netflix Customer Service.
